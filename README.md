# oauth2-pkce
🔐 Simple implementation of OAuth 2.1 Authorization Code Flow with PKCE. Built with best practices—no implicit flow, secure token handling, and aligned with the OAuth 2.1 draft spec.

# Getting Started
```sh
# Run Authorization server
cd ./server
pnpm install
pnpm run start:dev
```

```sh
# Run Client server
cd ./auth-client
pnpm install
pnpm run start:dev
```

```sh
# Run Frontend
cd ./client
pnpm install
pnpm run dev
```
